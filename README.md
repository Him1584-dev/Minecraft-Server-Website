# Minecraft-Server-Website
This is a template for a Minecraft Server Website. You can modify it freely, and there is no need to make it open source. All you need to do is to state the project link.
You only need to put it to your Web server,or edit it,then you can use it.

这是一个Minecraft服务器网页的模板。你可以随意修改它，修改后可以商用，也不必开源，仅需声明此项目链接即可。
你仅需将它们放置在你的网站服务器中，或者编辑一下，然后你就可以使用它。
