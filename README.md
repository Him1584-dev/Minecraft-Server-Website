# Minecraft-Server-Website
This is a template for a Minecraft server webpage. You can modify it as you wish. After modification, it can be used for commercial purposes without having to be open-sourced. All you need to do is keep the copyright notice at the bottom of the page.
You only need to place them on your website server, or make some edits, and then you can use it.

这是一个Minecraft服务器网页的模板。你可以随意修改它，修改后可以商用，也不必开源，仅需保留页面底部的版权声明即可。
你仅需将它们放置在你的网站服务器中，或者编辑一下，然后你就可以使用它。
